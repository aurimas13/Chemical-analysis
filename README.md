This is a simple project that includes two Jupyter files ([one about lipohilicities](https://github.com/aurimas13/Chemical-analysis/blob/main/physical_chemistry_lipophilicity.ipynb) & [the other about drawing structures](https://github.com/aurimas13/Chemical-analysis/blob/main/drawing_molecules.ipynb)) with data files removed from the repository. An analysis of lipophilicities & drawings are shown through notebooks that looks at lipophilicities of drug molecules and molecule drawings through RDKit.
